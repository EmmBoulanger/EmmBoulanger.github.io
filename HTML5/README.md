# About My Flare Project
This project is a multipage webpage detailing my understanding of web accessibility, and a team portion on how to use the **ClipChamp** software accessibly. Each section was split up into modules and, working in small teams, we worked to define these properties.

## Goals
1. Demonstrate an understanding of web accessibility and it's importance.
2. Work as a team to write a guide on using accessibile functions in **ClipChamp**.
3. Learn MadCap Flare.

## Teamwork
For this project we were split up into 6 groups. 
Each group was tasked with covering a specific topic on accessibility in **Clipchamp**.
### Team 3
* Arisa
* Mary
* Emma

Our module's purpose was to define integrated and audio description, describe how to use audio/integrated description in a video, and some best tips and practices. We divided our section into three parts, each writing our own section before peer editing for consistencies.

We then peer reviewed with the other groups, giving feedback and checking information.
This project will include all groups sections in the final website.

## Transferable Skills
* Collaboration
* Modular form of writing
* The **ClipChamp** software
